## Description

I turned a  multi-page [`openai.com`](https://openai.com/) site into a single page application.

## Project setup

```jsx
npm i
npm start
```

[http://localhost:3000](http://localhost:3000) to view it in the browser.

[http://localhost:3000](http://localhost:3000) - data.

## Stack

* React
* Redux
* TS
* Json-server
